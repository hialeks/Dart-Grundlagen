## Executable example

[`Dart-Grundlagen.dart`](Dart-Grundlagen.dart) is the German executable
companion to the three reference documents.

```sh
dart format Dart-Grundlagen.dart
dart analyze Dart-Grundlagen.dart
dart run Dart-Grundlagen.dart
```

The reference targets modern Dart 3. The example is verified with the Dart SDK
installed in this workspace.
